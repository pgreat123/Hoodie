# HOODIE

A premium AI-powered crypto analytics dashboard built with **HTML, CSS, and Vanilla JavaScript**.

## Features

- Live DexScreener integration
- Live token price
- Market Cap
- Liquidity
- 24H Volume
- Buy/Sell statistics
- AI Prediction Engine
- 5m / 15m / 1h / Daily / Monthly predictions
- Live Chart.js price chart
- Whale Activity panel
- Market Sentiment meter
- Prediction History
- Countdown timer
- Glassmorphism UI
- Animated particle background
- Fully responsive
- GitHub Ready
- Vercel Ready

## Deployment

### GitHub

1. Create a new repository.
2. Upload all project files.
3. Commit and push.

### Vercel

1. Import your GitHub repository.
2. Framework Preset: **Other**
3. Build Command: **None**
4. Output Directory: **/**
5. Deploy.
